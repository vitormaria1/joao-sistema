# Arquitetura Inicial

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS 4
- Supabase Auth + Postgres

## Módulos Prioritários

1. Autenticação e perfis
2. Dashboard do administrador
3. Área do aluno
4. CRM comercial
5. Gestão de tarefas
6. Relatórios e progresso

## Modelo de acesso inicial

- `admin`: João com controle total
- `operator`: apoio operacional futuro
- `student`: cliente que participa dos programas

## Estratégia de build

1. Fundar autenticação e perfis
2. Estruturar entidades centrais do negócio
3. Implementar experiência operacional do admin
4. Implementar experiência do aluno
5. Adicionar relatórios, automações e refinamentos
